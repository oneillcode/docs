## Local Development

Run Superblocks applications on your local machine for faster development and testing.

## Setup

Install the Superblocks CLI:

```bash
npm install -g @superblocks/cli
```

## Running Locally

Start the local development server:

```bash
superblocks dev
```

Your application will be available at `http://localhost:3000`
