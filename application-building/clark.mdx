Clark is Superblocks' AI-powered development assistant that helps you build applications faster.

## How to Use Clark

Simply type your request in natural language and Clark will help you:

```javascript
// Example: Ask Clark to create a data table
"Create a table showing all users from the database";

// Clark will generate the query and UI components for you
```
