## API Builder

Build and manage RESTful APIs directly within Superblocks.

## Creating APIs

<Steps>
  <Step title="Define Endpoint">
    Set the HTTP method (GET, POST, PUT, DELETE) and path
  </Step>
  <Step title="Add Logic">Write JavaScript to handle requests</Step>
  <Step title="Configure Response">
    Define the response format and status codes
  </Step>
  <Step title="Test">Use the built-in testing tools</Step>
</Steps>

## Example API

```javascript
// GET /api/users/:id
const userId = request.params.id;

const user = await Database.query(`
  SELECT * FROM users WHERE id = ${userId}
`);

return {
  status: 200,
  body: user,
};
```
