## What's Included

- **Core API**: Full REST API with authentication
- **Database Support**: PostgreSQL and MySQL compatibility
- **Authentication**: OAuth 2.0 and JWT support
- **Webhooks**: Real-time event notifications
- **Dashboard**: Web-based administration interface
