## APIs

Connect to any REST or GraphQL API:

```javascript
// REST API Example
const response = await RestAPI.request({
  url: "https://api.example.com/users",
  method: "GET",
  headers: {
    Authorization: `Bearer ${api_token}`,
  },
});
```
