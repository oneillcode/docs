## SOC 2 Type 2 Compliance

Superblocks has achieved SOC 2 Type 2 certification, demonstrating our commitment to security and compliance.

## What is SOC 2?

SOC 2 is a framework for managing customer data based on five Trust Service Principles:

<CardGroup cols={2}>
  <Card title="Security" icon="shield-halved">
    Protection against unauthorized access
  </Card>
  <Card title="Availability" icon="check-circle">
    System availability for operation
  </Card>
  <Card title="Processing Integrity" icon="gear">
    System processing is complete and accurate
  </Card>
  <Card title="Confidentiality" icon="eye-slash">
    Confidential information is protected
  </Card>
  <Card title="Privacy" icon="user-shield">
    Personal information is collected and handled properly
  </Card>
</CardGroup>

## Our Certification

<Info>

  auditors.
</Info>

### Type 2 vs Type 1

- **Type 1**: Point-in-time assessment
- **Type 2**: Continuous monitoring over time (typically 6-12 months)
