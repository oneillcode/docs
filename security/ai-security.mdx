## Sensitive Data

<Warning>
  Avoid including sensitive data in AI prompts: - API keys and credentials -
  Personal identifiable information (PII) - Proprietary business logic -
  Customer data
</Warning>

## Best Practices

```javascript
// ✅ Good: Generic request
"Create a user table with email and name fields";

// ❌ Bad: Contains sensitive data
"Create a table with these users: john@company.com, API key: sk_live_...";
```
